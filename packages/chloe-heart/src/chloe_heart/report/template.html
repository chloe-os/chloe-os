Methodology

Overview of each pipeline stage

  1. ECG Ingestion — Raw ECG signal data is loaded from the source file (CSV, EDF, or WFDB format) and validated for sample rate, duration, and signal integrity.
  2. Signal Preprocessing — Bandpass filtering removes baseline wander and high-frequency noise. QRS complexes are detected using adaptive threshold algorithms. R-R intervals are calculated and validated.
  3. Cardiac Analysis — Arrhythmia detection classifies each beat and identifies abnormal rhythms (PVCs, PACs, AFib, VTach, bradycardia, tachycardia, AV blocks). HRV metrics are computed in both time and frequency domains.
  4. Health Scoring — A composite cardiac health score (0–100) is calculated from five weighted components: rhythm regularity, HRV, heart rate, trend stability, and exercise tolerance. Breed-specific baselines are applied.
  5. Community Interpretation (optional) — AI-assisted analysis provides additional context, pattern recognition, and plain-language summaries of the cardiac findings.
  6. Report Generation — This self-contained HTML report is produced, combining all pipeline outputs into a format suitable for review by owners and veterinary cardiologists.
